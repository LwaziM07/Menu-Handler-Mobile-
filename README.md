# Menu-Handler-Mobile-
A mobile version of the application. Adapted from the system from JAVA to Typescript. A simple design but very functional.

# Changelog

## Version 1.0.0 (3 Oct)
* Set up the Home screen to integrate the core functionalities: Adding, displaying and saving menu items.

## Version 1.1.0 (6 Oct)
* Added a description Array to the Menu Edit Screen to allow detailed descriptions of menu items.

## Version 1.2.0 (18 Nov)
* Implemented a course filter screen and a menu edit screen that had been revamped.
* Completely overhauled the home screen code for better readability and functionality, displaying menu items alone and nothing else.
* Resolved a major bug causing the terminal to stop receiving commands and blocking commits to the master branch by adding the node_modules folder to .gitignore.
* I attempted to implement bottom tab navigation but switched to STack Navigator after experiencing terminal execution issues.
* Integrated Stack navigation for seamless navigation between screens, effectively passing props across the app.
* Updated the project's dependencies for better compatibility with the Expo Go app.

## Version 1.3.0 (19 Nov)
* Fully implemented Stack Navigation across all screens after challenges with Bottom Tabs navigation.
* Added functional components and defined props to pass data between screens (Home, Filter, and Menu edit screens).
* Added the item removal functionality on the Menu Edit screen, ensuring updates on the menu reflect immediately across all screens.
* Designed and styled :
  * Buttons for navigation between screens.
  * Images for the course filter and menu edit screens
* Improved the UI/UX with:
  * Revamped button designs.
  * Made user interaction across screens more intuitive.

## Version 1.4.0 (20 Nov)
* Added functionality to display menu item descriptions on all relevant screens.
* Integrated and styled icons for buttons, improving the app's appeal as a result.
* Implemented logic to calculate average course type prices:
  * Displayed these averages on the home screen.
  * Simplified price management across the different courses.
* Revamped navigation button design for a more intuitive user experience.

## Version 1.5.0 (21 Nov)
* Finalised the project by adding references to documentation
